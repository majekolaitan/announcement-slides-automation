# ⛪ SDA Kubwa Announcement System

**Automated PowerPoint Generation for Weekly Church Services**

This system takes raw announcement text (from WhatsApp, bulletins, or notes), processes it through AI, and automatically generates a professional, branded, and looping PowerPoint presentation.

---

## ✨ Features

- **One-Click Generation:** Double-click one file to build and open your presentation.
- **AI-Optimized:** Uses "punchy" fragments and SDA-appropriate emojis.
- **Hard-Coded Looping:** The output file is programmed to loop continuously (perfect for lobby screens or pre-service countdowns).
- **Smart Transitions:** Slides automatically advance every 8 seconds with a professional "Push Left" transition.
- **Branding Control:** Positioning, fonts, and colors are controlled via a standard PowerPoint Template (`SDA_Template.pptx`).

---

## 📂 File Structure

- **`run_announcements.bat`**: **The only file you need to run.** It sets up the environment and starts the process.
- **`build_slides.py`**: The "brain" of the system. It reads the data and builds the slides.
- **`announcements.json`**: The data file. This is where you paste the code generated by the AI.
- **`SDA_Template.pptx`**: Your master design. Edit this file to change the look of your slides.
- **`prompt.txt`**: The instructions you copy-paste into ChatGPT or Claude to process your text.

---

## 🛠️ One-Time Setup

1.  **Install Python:** Download from [python.org](https://www.python.org/).
    - _Crucial:_ Check the box that says **"Add Python to PATH"** during installation.
2.  **Template:** Ensure your `SDA_Template.pptx` is in the folder. If it is missing, the script will generate plain white slides instead.

---

## 🚀 Weekly Workflow (The 3 Steps)

### Step 1: Process with AI

1.  Open `prompt.txt` and copy the entire text.
2.  Go to [ChatGPT](https://chat.openai.com) or [Claude](https://claude.ai).
3.  Paste the prompt, and at the bottom, paste your **raw announcement text**.
4.  The AI will reply with a block of code (JSON). **Copy that code.**

### Step 2: Update the Data

1.  Open `announcements.json` (Right-click > Open with Notepad).
2.  Delete everything inside.
3.  Paste the code you copied from the AI.
4.  **Save and Close** the file.

### Step 3: Generate the Slides

1.  Double-click **`run_announcements.bat`**.
2.  Wait a few seconds. The script will:
    - Check for required libraries.
    - Create a professional PowerPoint.
    - Automatically open the final presentation for you.
3.  Press **F5** to start the show!

---

## 🎨 Customizing the Design

You do **not** need to edit the Python code to change the design.

1.  Open `SDA_Template.pptx`.
2.  Go to **View > Slide Master**.
3.  Edit the **second slide** (the "Title and Content" layout).
    - Change the background, font color, or logo.
    - **Tip:** Ensure the "Body" text box is set to "Shrink text on overflow" so long announcements fit automatically.
4.  Save the template. Every slide generated from now on will use your new design.

## 🔍 Troubleshooting

| Issue                              | Solution                                                                                           |
| :--------------------------------- | :------------------------------------------------------------------------------------------------- |
| **Window closes immediately**      | Open a Command Prompt, drag the `.bat` file into it, and press Enter to see the error message.     |
| **"Python not found"**             | Re-install Python and make sure to check "Add Python to PATH."                                     |
| **Permission Denied**              | Close the PowerPoint file `SDA_Kubwa_Announcements.pptx` if it is currently open.                  |
| **JSON Error / Generation Failed** | Ensure you copied the _entire_ block of code from the AI, including the `[` and `]` brackets.      |
| **Emojis are Black & White**       | Ensure you are running on Windows 10 or 11. The script is optimized for the "Segoe UI Emoji" font. |
